# sqlite-bookstore-database-project
SQLite Bookstore Database Project
Overview

This project demonstrates the design and implementation of a relational database system for an online bookstore using SQLite, Python, and Pandas.

It covers database schema design, data insertion, SQL querying, and analytical reporting using Pandas integration.

Database Schema

The database bookstore.db contains three main tables:
Books
Customers
Orders

With:
Primary Keys
Foreign Key relationships
Proper constraints (NOT NULL, UNIQUE, DEFAULT)
